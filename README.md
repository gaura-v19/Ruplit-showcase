# Ruplit-showcase

Ruplit is a peer-to-peer expense-splitting app built with Spring Boot, PostgreSQL, Redis, and React Native. Implements JWT authentication, Redis-based idempotency, and optimistic locking for concurrent transaction safety. Uses the Saga pattern for distributed transaction management and a greedy algorithm for optimal debt settlement. Deployed on Render, Neon, and Upstash for a fully functional production environment.
